Archivos para Search Console

1) Subí ambos archivos a la raíz del sitio:
- sitemap.xml
- robots.txt

La raíz es el mismo nivel donde está index.html.

2) En Search Console:
- ir a Sitemaps
- enviar: sitemap.xml

3) Luego usar URL Inspection para pedir indexación de:
- https://surco.lat/
- https://surco.lat/blog/
- https://surco.lat/blog/cuaderno-de-campo-digital-dia-a-dia/
- https://surco.lat/blog/registrar-pulverizaciones-whatsapp/
- https://surco.lat/blog/registro-fitosanitario-que-datos-guardar/
- https://surco.lat/blog/historial-de-labores-por-lote/
